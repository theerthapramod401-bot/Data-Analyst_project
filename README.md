# Data-Analyst_project
# Multi-Touch Marketing Attribution & ROI Dashboard

## Project Overview
This project aims to analyze marketing campaign performance across multiple channels and evaluate their contribution to customer conversions and return on investment (ROI). The objective is to build an interactive dashboard that provides insights into campaign effectiveness, channel performance, customer engagement, and marketing ROI.

---

## Dataset Information

### Dataset Name
Marketing Campaign Performance Dataset

### Dataset Size
- Records: 200,000
- Features: 16

- Duration
- Channel_Used
- Conversion_Rate
- Acquisition_Cost
- ROI
- Location
- Language
- Clicks
- Impressions
- Engagement_Score
- Customer_Segment
- Date

---

## Project Objectives
- Analyze marketing campaign performance across multiple channels.
- Measure and compare Return on Investment (ROI).
- Identify high-performing marketing channels.
- Evaluate customer engagement and conversion trends.
- Generate actionable insights through data visualization.
- Develop an interactive dashboard for decision-making.
